[ 언어 와 언어 사이 ] : 번역이 필요, 컴파일/인터프린터/랜더링
[ HTML언어 와 기계어 사이 ] : 브라우저( 크롬,엣지,사파리 등등)
[ HTML 언어]
    1. H(Hyper: 이동) T(text:텍스트) M(markup:마크업) L(Language:언어)
        이동이 가능한 문자들을 제공하는 마크업언어
    2. 마크업 : 구조화된 표식
    3. 언어 : 규칙/가호 집합
        마크업 언어란 : 텍스트 압 뒤를 마크/덮쳐서 하는 언어
            * 각 마크업에는 미리 정해진 기능들이 포함
        ex <h1>안녕</h1>
    4. 주석이란 : 메모지(랜더링 안된 구역)
        <!-- 내용물 -->
    [ 마크업 ]
        - HTML 언어를 만든 회사에서 미리 정의한 규칙 = 문법
        1. <마크업명> 마크대상 </마크업명>
                - <h1> </h1>
        2. <마크업명/>
            - <img />
            
        - 각 마크업들은 서로 다른 속성(본질의 특성)을 갖는다.
            방법1. < 마크업명 속성명="값">
            - <나라는 존재 NAME"이름" JOB="강사" >
            - HTML은 프로그래밍 언어 일까? [X] 마크업 언어 ---> REACT ( JS프로그래밍)


    [ 마크업 종류]
        *마크업 작성 주의할점 : 마크업 안에 포함되는 마크업 작성시에는 들여쓰기(TAB) 한다(상속관계).
            부모 
        1. <HTML> : HTML 모든 마크업 전체를 감싼 마크업
        2. <head> : HTML 문서의 옵션 정보 마크업 감싼 마크업
        3. <title> : HTML 문서의 페이지 제목
        4. <link href="css파일경로 > : html 문서의 css포함  
        5. <script src="JS파일경로 > : HTML 문서의 JS포함   
        6. <body> : html 문서의 본문 
        7. <H1> ~ <H6> : 제목(head) 마크업 , 글자크기 
        8. <p> </p> : 문단(Parageraph) 마크업 , 구역 의미
        9. <br /> : break rule, 줄바꿈 의미
        10. <hr /> : horiztontal rule, 가로선 의미
        11. <ol> : order list , 순서(번호) 있는 목록
        12. <ul> : unorder list , 순서(번호)없는 목록
        13. <li> : ol 또는 li 마크업안에 포함되는 마크업으로 목록의 항목
        14. <a> : html/페이지 이동
           - <a href="이동할경로" > 텍스트 </a> 
           - <a href="이동할경로" target="_blank" > 텍스트 ,/a> 새로운 탭/창 열기

        15. <img /> : 이미지 마크업
        16. <audio> </audio> : 오디오 마크업
        17. <video> </video> : 비디오 마크업
            15~16 공통 속성 , src="파일경로"
            16~17 공통 속성, 
                controls="controls" : 컨트롤러 제공
                autoplay="autoplay" : 자동실행,정책상 차단
                muted="muted" :음소거, autoplay + muted : 비디오 자동 실행 가능 오디오 안됨 

        18. <input> : 입력 상자 마크업, 사용자로 부터 입력받을수 있는 상태 
                - [ 주요 속성명 ]
                (1) type        : 입력 방법 
                    type = "text" : 문자 입력 , type 생략시 기본값
                    type = "password" :가려진 문자 입력
                    type = "file" : 첨부파일 * multiple 다중선택
                    type = "radio" : 단일 선택 상자 * name 속성값이 동일
                    type = "checkbox" : 복수선택상자
                    type = "number" : 숫자 입력
                    type = "date" : 날짜 선택
                    type = "datetime-local : 날짜/시간 선택
                    type = "time" : 시간 선택
                    type = "month" : 월 선택

                (2) name        : 동일한 마크업명 간의 식별 이름
                (3) value       : 입력상자에 데이터
                (4) disabled    : 비활성화
                (5) readonly    : 수정불가능(수정금지)
                (6) maxlength   : 최대 글자수 제한 ex) 5 5글자만 됨
                (7) autofocus   : 지정한 입력상자의 포커스(마우스커서) 설정
                (8) placeholder : 입력상자의 가이드 텍스트

                20. 표 만들기
                <table> 전체 표를 감싼 마크업
                    <tr> 테이블행/가로 
                        <th> : 테이블 제목(열/세로) 셀/칸
                        <td> : 테이블 데이터(열/세로) 셀/칸

                속성
                    boarder = "1" : 테이블 테두리
                    style="border-collapse : collapse;" : 이중 테두리를 하나의 테두리로 변경
                    rowspan = "행병합"
                    colspan = "열병합"
